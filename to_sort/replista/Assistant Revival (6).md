# Tutorial: Getting Started with Assistant Memorial Edition

  

\*\*Difficulty\*\*: Beginner

\*\*Time\*\*: 15-20 minutes

\*\*Prerequisites\*\*: None

  

## Learning Objectives

  

By the end of this tutorial, you will be able to:

\- Navigate the Assistant interface

\- Send your first message

\- Upload and reference files

\- Create a checkpoint

\- Use basic assistant prompts

  

## Step 1: Understanding the Interface

  

When you first open Assistant Memorial Edition, you'll see three main areas:

  

\`\`\`

┌─────────────┬──────────────────────┬──────────────┐

│ Sidebar │ Chat Messages │ File List │

│ │ │ │

│ - Files │ Conversation here │ Uploaded │

│ - Prompts │ │ files │

│ - Settings │ │ │

└─────────────┴──────────────────────┴──────────────┘

\`\`\`

  

\*\*Key Areas:\*\*

\- \*\*Left Sidebar\*\*: Navigation and settings

\- \*\*Center Panel\*\*: Chat conversation

\- \*\*Right Panel\*\*: File management

  

## Step 2: Send Your First Message

  

1. Click in the text input at the bottom of the chat panel

2. Type a simple question:

\`\`\`

What is JavaScript?

\`\`\`

3. Press \*\*Enter\*\* or click the \*\*Send\*\* button

  

\*\*Expected Result\*\*: The assistant will respond with an explanation about JavaScript.

  

## Step 3: Upload a File

  

Let's upload a code file to get help with it.

  

1. Click the \*\*Upload\*\* button in the right panel (or drag and drop)

2. Select a \`.js\`, \`.py\`, \`.html\`, or any code file

3. Wait for the upload to complete - you'll see it appear in the file list

  

\*\*Pro Tip\*\*: You can upload multiple files at once!

  

## Step 4: Reference Files in Messages

  

Now that you have a file uploaded, you can reference it in your questions.

  

1. In the chat input, type \`@\` to see a list of uploaded files

2. Select your file from the list (or type the filename)

3. Complete your question:

\`\`\`

@myfile.js Can you explain what this code does?

\`\`\`

4. Send the message

  

\*\*What Happens\*\*: The assistant will read the file content and provide a detailed explanation.
